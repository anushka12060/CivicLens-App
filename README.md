# CivicLens

### AI-Powered Hyperlocal Civic Issue Reporting & Resolution Platform

Built for the **Vibe2Ship Hackathon**  
**Coding Ninjas × Google for Developers**

---

## Overview

CivicLens is an AI-powered platform that helps citizens report, track, and resolve local civic issues such as potholes, broken streetlights, waterlogging, garbage accumulation, and damaged roads.

Powered by **Google Gemini 3.5 Flash**, the platform analyzes uploaded images, classifies issues, estimates severity, assigns the responsible department, prioritizes reports, and assists authorities with AI-generated resolution plans.

---

## Features

-  AI-powered image-based issue reporting using Gemini Vision
-  4-Agent AI pipeline (Vision, Classification, Priority & Resolution)
-  Real-Time Priority Dispatch Board
-  Interactive Issue Map using Leaflet
-  Ward Health Index with AI-generated insights
-  Gemini Resolution Agent with contextual AI chat
-  Community verification through upvotes
-  Civic Points gamification system

## 🚀 Running Locally

### Clone the repository

```bash
git clone https://github.com/anushka12060/CivicLens-App.git
```

### Navigate to the project

```bash
cd CivicLens-App
```

### Install dependencies

```bash
npm install
```

### Create a `.env.local` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the development server

```bash
npm run dev
```
